---

**Part I : Basics **

- Origin
- Version history
- Requirements and dependencies
  - Runtime environment
  - Package manager
- Working mechanism
- Programming Paradigm
  - Statically vs dynamically typed
  - Primitives and reference types
  - Imperative (Procedural, Structured, Object Oriented) vs declarative (Functional, Logic)
- Setup & installation

**Part II : Core **

- Syntax
  - Variables
  - Arrays
  - Objects
  - Functions
  - Classes
  - Networking
  - Making API calls
  - Asynchronicity
  - Adding/generating libraries
- OOP Principle Implementation
  - Encapsulation
  - Inheritence
  - Polymorphism
  - Abstraction
